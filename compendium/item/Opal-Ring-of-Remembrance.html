Opal Ring of RemembranceLevel 29 Rare

The large fire opal set into this bauble flares brightly when it bestows mental clarity.

Ring Slot        2,625,000 gp

Property

Gain a +2 item bonus to Intelligence attacks, and a +4 item bonus to Intelligence checks and Intelligence-based skill checks.

Power Daily (Minor Action)

Regain the use of an arcane encounter utility power that you have already used (as if you hadn't used it this encounter).
    If you've reached at least one milestone today, you can instead regain the use of an arcane daily utility power (as if you hadn't used it today).



Published in Adventurer's Vault, page(s) 157.
