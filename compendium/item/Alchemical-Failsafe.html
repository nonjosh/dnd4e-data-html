Waist Slot (embedded component)

Requirement: You must have the living construct racial trait to use this item.

Property

You can stow one consumable alchemical item within this item.

Power

When you use your second wind, if you have a consumable alchemical item stowed that is 10th level or lower, you can use the stowed item as a part of the same action without having to draw it or spend an action to use it.

Level 16: You can use the at-will power with alchemical items of 20th level or lower.

Level 26: You can use the at-will power with any alchemical item.
